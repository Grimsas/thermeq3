# Alpha version for Yún Rev2

Just for test issues.

## How to get install script
`wget --no-check-certificate --quiet -O /root/upgrade.sh https://raw.githubusercontent.com/autopower/thermeq3/master/install/alpha_Yun_Rev2/install.sh;chmod +x /root/install.sh`
After download please run `/root/install.sh`, script downloads thermeq3 version and interactive config which will be run after succesfull download.
